# 第一部分 数据管理系统的功能、模型与语言

本部分介绍数据管理系统的基本概念、发展历史、数据模型和访问语言。

## 学习要点

### 通用知识
- 理解数据管理系统的基本概念和架构
- 了解数据管理系统的发展历史
- 掌握不同数据模型的特点和应用场景
- 理解数据访问语言的作用和分类

### 关系数据库系统
- 深入理解关系模型的理论基础
- 掌握SQL语言的基本语法和使用方法
- 学习数据定义、操纵和查询操作
- 理解视图的概念和作用

### 文档数据库系统
- 理解文档模型的特点
- 掌握文档数据库查询语言的使用

### 图数据库系统
- 理解图模型的基本概念
- 掌握图结构的表示方法
- 学习图模式匹配的基本操作
- 了解图数据库查询语言
