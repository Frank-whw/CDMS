# 第三部分 查询处理功能

本部分介绍数据管理系统的查询处理功能，包括查询处理的基本原理和优化技术。

## 学习要点

### 通用知识
- 理解数据计算的基本概念
- 掌握查询处理的一般流程

### 关系数据库系统
- 深入了解关系数据库的查询处理机制
- 掌握关系算子的实现方法和代价估算
- 学习关系数据库的查询优化技术

### 文档数据库系统
- 理解文档数据库的查询处理特点

### 图数据库系统
- 掌握图数据库的查询处理方法
- 学习图数据库的查询优化策略
- 了解图数据库中的基础和专用图算法
